## ⚙️ OPERATIONS
**Forum:** `operations`  
**Purpose:** Execution of treasury, administrative, and governance operations linked to active projects.

**Tags:**  
`[Treasury]` `[Finance]` `[Admin]` `[DevOps]` `[Governance]` `[Documentation]` `[Discussion]` `[Priority]` `[Task]`

**Usage Guide:**  
- **[Treasury]** — Fund transfers, multisig actions, or audits.  
- **[Finance]** — Budgeting, reporting, or financial analysis.  
- **[Admin]** — Permissions, access, or workflow maintenance.  
- **[DevOps]** — Automation, bots, and technical systems.  
- **[Governance]** — Implementation of approved proposals.  
- **[Documentation]** — Final records or completion reports.  
- **[Discussion]** — Open coordination or planning threads.  
- **[Priority]** — Time-sensitive or high-impact tasks.  
- **[Task]** — Actionable items for execution.

**Example Workflow:**  
1. Governance approves funding for `#Baobab`.  
2. Operations opens `[Treasury][Task] Execute Multisig Payout for #Baobab.`  
3. Posts TX link and summary.  
4. Tags `[Documentation]` when finalized.

## 🏁 RACETRACK (Project Hub)
**Forum:** `racetrack`  
**Purpose:** Central hub for all DAO projects. Each thread = one project.

**Posting Rules:**  
- Only **mods and bots** can create threads.  
- Threads use the **RaceTrack Project Template** below.  
- Comments used for mod/bot status updates only.

**Tags:**  
`[In Progress]` `[On Hold]` `[Complete]` `[Archived]` `[Retro]` `[Funding]` `[Marketing]`

**RaceTrack Template:**
```
### 🏁 Project Title: Baobab — Run to Mainnet

**Summary:**  
Short description of the project and objectives.

**Forums Involved:**  
#operations — Treasury setup  
#technologia — Smart contract deployment  
#marketing — Launch campaign  
#bizdev — Partnerships and sponsors  

**Tags:** [In Progress] [Funding]

**Links:**  
📊 Snapshot → [link]  
💸 Multisig → [link]  
🧱 GitHub → [link]  
🎯 Campaign → [link]

**Lead:** @username  
**Start Date:** YYYY-MM-DD  
**Expected Completion:** YYYY-MM-DD
```
